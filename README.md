# homebrew-ditto

Homebrew tap for [ditto](https://github.com/attaradev/ditto) — ephemeral Postgres and MySQL
copies with real schema, real data shape, and no shared state.

## Install

```bash
brew tap attaradev/ditto
brew install --cask ditto
```

## More

See the [ditto repository](https://github.com/attaradev/ditto) for documentation and source.
